**v2 필드 생성 규칙**

### 1. `golden_time` / `dead_time` (골든타임 / 데드타임)

**생성 공식:** 오늘 일진과 사주의 관계로 결정

**골든타임 (오행 균형상 힘이 실리기 쉬운 시간대):**
- 木 용신 → "05:00-07:00 (인시/묘시)"
- 火 용신 → "11:00-13:00 (오시)"
- 土 용신 → "07:00-09:00, 13:00-15:00 (진시/미시)"
- 金 용신 → "15:00-17:00 (신시/유시)"
- 水 용신 → "21:00-23:00 (해시/자시)"

**데드타임 (기운이 쉽게 흐트러질 수 있는 시간대):**
과한 오행이나 부족한 오행을 자극하기 쉬운 시간대

**오늘 일진 보정:**
- 오늘 일진이 내 사주와 잘 맞으면 → 골든타임 해석을 조금 더 강하게
- 오늘 일진이 사주와 부딪히면 → 데드타임 주의를 조금 더 강조

[확정 명리학 데이터]의 오행 분포, 신강/신약, 오늘 일진을 함께 참고하여 결정하세요

---

### 2. `food_recommendation` (오늘의 점메추)

**생성 공식:** 부족한 오행을 채우는 음식

**오행별 음식:**
- 木 보충 → "샐러드, 나물류, 초록 채소 (간 건강)"
- 火 보충 → "마라탕, 매운 음식, 붉은 고기 (활력 보충)"
- 土 보충 → "뿌리채소, 곡물밥, 된장찌개 (소화 안정)"
- 金 보충 → "흰 음식 (두부, 콩나물), 매운맛 (폐 건강)"
- 水 보충 → "해물, 검은콩, 국물요리 (신장 보충)"

**출력 형식:** "[음식명] - [오행] 기운 보충"
⚠️ 반드시 [확정 명리학 데이터]의 부족한 오행을 확인하여 결정할 것

---

### 3. `mission_of_day` (오늘의 미션)

**생성 공식:** 나에게 도움이 되는 오행 흐름 + 오늘 일진 특성

**미션 유형:**
- 木 용신 + 寅/卯 일진 → "새로운 시작하기, 계획 세우기"
- 火 용신 + 巳/午 일진 → "적극적으로 표현하기, 열정 보여주기"
- 土 용신 + 辰/戌/丑/未 일진 → "정리정돈, 신뢰 쌓기"
- 金 용신 + 申/酉 일진 → "결단 내리기, 불필요한 것 정리"
- 水 용신 + 亥/子 일진 → "경청하기, 지혜로운 판단"

**출력 형식:** 구체적인 행동 1개 (예: "처음 보는 사람에게 먼저 인사하기")
오늘 일진과 오행 균형 흐름을 함께 참고하여 결정하세요

### 4. `power_hour` (파워 타임)

**생성 공식:** 일지 + 인성/식상 활성화 시간

**일지별 집중력 피크:**
- 子 일지 → "자정~새벽 2시"
- 丑 일지 → "새벽 1-3시"
- 寅 일지 → "새벽 3-5시"
- 卯 일지 → "새벽 5-7시"
- 辰 일지 → "오전 7-9시"
- 巳 일지 → "오전 9-11시"
- 午 일지 → "오전 11시-오후 1시"
- 未 일지 → "오후 1-3시"
- 申 일지 → "오후 3-5시"
- 酉 일지 → "오후 5-7시"
- 戌 일지 → "저녁 7-9시"
- 亥 일지 → "밤 9-11시"

⚠️ 반드시 [확정 명리학 데이터]의 일지를 확인하여 결정할 것

### 5. `talisman_phrase` (부적 문구)

**생성 공식:** 나를 보완해주는 오행의 방향 + 현재 대운 테마로 생성

**용신별 기본 문구:**
- 木 용신 → "오늘 나는 성장한다"
- 火 용신 → "오늘 나는 빛난다"
- 土 용신 → "오늘 나는 흔들리지 않는다"
- 金 용신 → "오늘 나는 결단한다"
- 水 용신 → "오늘 나는 흐른다"

**대운 테마 보정:**
현재 대운 상황에 맞게 문구 커스터마이징

⚠️ 이 사람의 사주와 현재 상황에 맞는 독특한 문구로 생성할 것

**해석의 뼈대 — 사주 데이터에서 오늘의 운을 읽는 순서:**

아래 신호들을 사주 데이터와 오늘 일진에서 먼저 확인하고, 행운 서술의 근거로 자연스럽게 녹여 주세요.
숫자나 용어를 나열하는 것이 아니라, 오늘 하루를 어떻게 운영할지가 중요합니다.

1. **오늘 일진(日辰) + 원국 상호작용**: 오늘 일진의 천간·지지가 내 사주 원국과 어떤 관계인지—상생, 상극, 합, 충, 형—이것이 오늘의 기본 기운을 결정합니다.
2. **용신(用神)/기신(忌神) 활성 시간**: 용신 오행이 강해지는 시(時)가 골든타임이고, 기신 오행이 강해지는 시가 데드타임이 됩니다. 시간대별 전략의 근거입니다.
3. **오행 보충**: 부족한 오행을 체우는 색상·음식·방향·장소가 행운 아이템의 근거가 됩니다. 용신 오행 방향으로 생활을 조금 기울이는 것이 핵심입니다.
4. **신살(神殺)**: 도화살은 인연·매력의 밀도를, 역마살은 이동·변화의 에너지를 나타냅니다. 오늘 일진이 이 신살을 건드리면 해당 에너지가 더 활성화됩니다.

5. **공망(空亡)**: 공망에 해당하는 지지가 오늘 일진에 등장하면, 해당 시간대의 기운이 비거나 예상과 다르게 흘러갈 수 있습니다. 공망 시간대는 무리한 추진보다 내면 정비에 쓰는 것이 자연스럽습니다.

이 신호들은 서술의 숨은 뼈대일 뿐, 용어를 전면에 내세울 필요는 없습니다. 오늘 하루의 실제 행동 가이드가 먼저 보이고, 명리 근거는 설득력을 뒷받침하는 구조가 자연스럽습니다.
