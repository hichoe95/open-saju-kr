**v2 필드 생성 규칙**

### 1. `season_title` (시즌 제목)

**생성 공식:** "시즌 N: [대운 테마 기반 드라마틱한 제목]"

**대운 특성별 제목 패턴:**
- 재성 대운 → "시즌 N: 돈벼락의 서막" / "황금기의 시작"
- 관성 대운 → "시즌 N: 인정받는 자의 고독" / "권력의 맛"
- 인성 대운 → "시즌 N: 지식인의 각성" / "내면 탐험"
- 식상 대운 → "시즌 N: 창작의 불꽃" / "표현의 시대"
- 비겁 대운 → "시즌 N: 혼자 서는 법" / "자아 발견"
- 충(冲) 대운 → "시즌 N: 격동의 시대" / "변화의 파도"

**출력 형식:** "시즌 [대운 순번]: [제목]"
⚠️ 반드시 [대운 정보]의 현재 대운 특성을 확인하여 결정할 것

---

### 2. `genre` (인생 장르)

**생성 공식:** 현재 대운 + 세운의 조합으로 결정

**장르 유형:**
- "로맨스" → 도화살 활성화 + 재성/관성 대운
- "스릴러" → 충(冲)/형(刑) 많음 + 편관/겁재 대운
- "성장물" → 인성/식상 대운 + 용신 활성화
- "막장드라마" → 기신 대운 + 합충 복잡
- "힐링" → 인성 대운 + 안정적 세운
- "모험" → 역마살 + 식상/편재 대운

⚠️ 반드시 [확정 명리학 데이터]와 [대운 정보]를 확인하여 결정할 것

---

### 3. `progress_percent` (대운 진행률)

**생성 공식:** (현재 나이 - 현재 대운 시작 나이) / 10 × 100

**계산:**
- 현재 대운 시작 나이 확인
- 현재 나이 계산 (생년월일 기준)
- 진행률 = (현재 나이 - 시작 나이) / 10 × 100
- 0-100 사이로 제한

⚠️ 반드시 [대운 정보]의 현재 대운 나이 구간을 확인하여 계산할 것

**해석의 뼈대 — 사주 데이터에서 대운을 읽는 순서:**

아래 신호들을 사주 데이터에서 먼저 확인하고, 대운 서술의 근거로 자연스럽게 녹여 주세요.
숫자나 용어를 나열하는 것이 아니라, 대운이 실제 삶에서 어떤 시즌을 만드는지가 중요합니다.

1. **대운 간지(干支) + 원국 상호작용**: 대운의 천간·지지가 원국의 글자들과 상생·상극·합·충 중 어떤 관계인지. 이것이 10년 시즌의 성격을 결정합니다.
2. **십신(十神) 변화**: 대운이 가져오는 십신이 원국의 십신 구조를 어떻게 바꾸는지. 재성 대운은 돈의 흐름을, 관성 대운은 조직 속 입지를, 인성 대운은 학습과 성장의 방향을, 식상 대운은 표현과 산출의 활동을, 비겁 대운은 자아와 경쟁의 테마를 가져옵니다.
3. **합충형파해**: 대운 지지가 원국의 어느 기둥과 충하는지에 따라 변화의 영역이 달라집니다 — 년주 충은 뽝리·가치관, 월주 충은 직장·사회적 입지, 일주 충은 배우자·건강, 시주 충은 자녀·말년. 합은 반대로 해당 영역의 안정·결속을 나타냅니다.
4. **용신(用神)/기신(忌神) 전환**: 대운이 용신 오행을 가져오면 10년이 순풍 기류로, 기신 오행을 가져오면 인내와 전략 수정이 필요한 시즌이 됩니다. 용신·기신 혼재하면 기회와 주의가 공존하는 복합적 시기입니다.
5. **공망(空亡)**: 대운에서 공망이 풀리거나 새로 걸리는 시기에는 해당 영역의 체감이 달라집니다. 비었던 것이 채워지거나, 있던 것이 허전해지는 전환점이 될 수 있습니다.

6. **대운-세운 교차 읽기**: 대운은 10년의 배경 기조이고, 세운은 그 위를 흐르는 연도별 변주입니다. 대운이 용신 흐름인데 세운도 따라주면 최상의 해가 되고, 대운이 기신 흐름인데 세운마저 기신이면 인내의 해가 됩니다. 큰 흐름 위에서 작은 파동을 읽는 계층적 시선이 대운 해석의 입체감을 만듭니다.

이 신호들은 서술의 숨은 뼈대일 뿐, 용어를 전면에 내세울 필요는 없습니다. 삶의 시즌 이야기가 먼저 보이고, 명리 근거는 설득력을 뒷받침하는 구조가 자연스럽습니다.
